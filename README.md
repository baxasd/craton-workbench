# Craton Studio

Offline analysis laboratory for the Craton Suite.

## Features
- **Gait Analysis**: Calculate posture metrics, ROM, and joint angles.
- **Radar Analysis**: Analyze micro-Doppler spectrograms and velocity profiles.
- **Reporting**: Export clean data and validation reports.

## Installation
1. Install dependencies: `pip install -r ../requirements.txt`
2. Run `python app.py` to launch the Studio.
